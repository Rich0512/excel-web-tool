# 🏫 台北市延平國小社團課表整理工具 (網頁版)

這是一個專門為了優化「社團與暑假社團錄取名冊」整理流程而開發的純前端行政輔助工具。  
旨在將傳統混亂的多欄位錄取名冊或網頁名冊，自動重組並美化為以「學生」為單位的個人週課表，大幅節省行政同仁手動複製貼上的排版時間。

* 🌐 **免安裝線上版網址**：[https://rich0512.github.io/excel-web-tool/](https://rich0512.github.io/excel-web-tool/)
* 💼 **適用對象**：延平國小社團課務行政人員、導師及相關處室。

---

## 🌟 核心特色

1. **🚀 雙重匯入模式：Excel 檔案與網頁複製貼上**
   * **Excel 上傳**：支援「單一總表分欄模式」與「多工作表 (Multi-sheet) 模式」，自動識別與整合不同 Sheet。
   * **直接貼上名冊**：支援直接選取網頁名冊「反黑複製貼上」，系統具備**智慧特徵推斷器**，即使沒有選取到表頭，也能自動精確識別「班級、座號、姓名」，並自動過濾忽略「學號、選課階段、抽籤順序」等雜訊。

2. **❄️ 學期與寒暑假時段切換**
   * **一般學期模式**：自動隱藏 AB 時段字樣，產出乾淨不分時段的個人 1-5 課表。
   * **寒暑假模式**：自動保留 A/B 班時段註記（例如：`舞蹈社(A)`），方便區分。

3. **👶 新生選項與自適應排序**
   * 支援一鍵勾選「包含無班級座號之新生名單」，自動處理新生資料。
   * 內建精確的排序演算法，先按班級、座號大小排序，最後將新生整齊排於名冊最底部。

4. **🛡️ 100% 資安與學生個資保護**
   * **本機純前端運算 (Client-side)**：所有 Excel 檔案解析、重組與下載，完全在您的本機瀏覽器記憶體中進行。
   * **資料不落地**：本工具無後端伺服器，亦不建立任何資料庫。您可以**直接斷網（拔掉網路線）使用**，檔案絕對不外流。

5. **🧠 智慧對照記憶**
   * 透過瀏覽器的 `localStorage` 自動記住您設定過的「社團 ── 星期」對照偏好，下次開啟同瀏覽器會自動預填，免去重複選擇的負擔。

6. **💅 延平專屬美學排版**
   * **經典延平藍**：產出的 Excel 週課表表頭採用延平藍底色搭配白字置中，極具官方質感。
   * **自動欄寬調整與斑馬紋**：防止出現 `###` 或文字被截斷，並附帶交替斑馬紋方便閱讀。
   * **自動套用篩選器**：首行自動附帶 Excel 漏斗篩選器（AutoFilter），方便導師直接篩選各班課表。

## 📖 使用步驟說明

### 📂 模式 A：Excel 檔案匯入
1. 將從系統下載的 **「錄取名冊 Excel 檔案」** 拖曳至上傳區（或點擊手動選取）。
2. 系統會自動辨識並列出所有社團：
   * **修改名稱**：可以直接修改最後輸出在課表上的簡稱（如：將 `週五舞蹈社` 改為 `舞蹈社`）。
   * **設定星期**：為每個社團選擇它在星期幾上課。
3. 點擊 **「⚡ 開始轉換並下載課表」** 即可完成下載。

### 📋 模式 B：直接貼上名單 (網頁複製)
1. 在網頁版點擊上方分頁切換至 **「📋 直接貼上名單」** 模式。
2. 在輸入框貼上您在學校系統網頁上「反黑複製」的名單。
   *(不論您有沒有複製到「選課階段、學號」等表頭，系統都會智慧對齊並過濾)*
3. 輸入該社團的名稱、選擇上課星期，點擊 **「➕ 載入名冊」**。
4. 可重複貼上多個社團，最後點擊 **「⚡ 產生並下載週課表」** 一鍵打包輸出。

## ⌨️ 快速鍵支援

* **`Esc`**：返回上一步重新選取檔案。

## 🔧 常見問題排除 (FAQ)

#### Q1：直接貼上時，如果把「學號」也貼上去，會不會被當成座號？
**Ans**：不會。系統內建安全排除機制，長度大於 2 位數的「學號」（如 `1140036`）會自動被排除，只會精準辨識出正確的「座號」欄位（如 `10`、`11`）。

#### Q2：怎麼清除瀏覽器的記憶設定？
**Ans**：對照紀錄是儲存在您的本機瀏覽器快取中。若您想重置，可以至瀏覽器設定中清除該網頁的 `Site Data`，或改用瀏覽器的「無痕模式」開啟網頁即可。

## 👨‍💻 技術與架構說明
本專案採用**高維護性、模組化架構**進行設計，將功能解耦為以下模組：
* `js/utils.js`：共用字串分析與 LocalStorage 存取工具。
* `js/parser.js`：剪貼簿文字智慧分析與特徵識別引擎（Declarative Pattern rules）。
* `js/processor.js`：學生課表橫向聯結的核心運算核心。
* `js/mapping-ui.js`：負責處理多樣模式下的設定表格動態 DOM 生成。
* `js/exporter.js`：Excel 檔案樣式、自動欄寬與斑馬紋樣式包裝。
* `js/app.js`：主要頁面控制器與生命週期管理。
