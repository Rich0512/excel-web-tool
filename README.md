# 🏫 台北市延平國小社團課表整理工具 (網頁版)

這是一個專門為了優化「暑假社團錄取名冊」整理流程而開發的純前端行政輔助工具。  
旨在將傳統混亂的多欄位錄取名冊，自動重組並美化為以「學生」為單位的個人週課表，大幅節省行政同仁手動複製貼上的排版時間。

* 🌐 **免安裝線上版網址**：[https://rich0512.github.io/excel-web-tool/](https://rich0512.github.io/excel-web-tool/)
* 💼 **適用對象**：延平國小社團課務行政人員、導師及相關處室。

## 🌟 核心特色

1. **🚀 零安裝，免去防毒阻擋**
   * 純網頁版應用，只要能上網就能使用。
   * 跨平台相容（Windows、Mac、iPad 均可）。
   * 絕不會被校園端防毒軟體（如趨勢防毒、Windows Defender）誤判攔截。

2. **🛡️ 100% 資安與學生個資保護**
   * **本機純前端運算 (Client-side)**：所有 Excel 檔案解析、重組與下載，完全在您的本機瀏覽器記憶體中進行。
   * **資料不落地**：本工具無後端伺服器，亦不建立任何資料庫。您可以**直接斷網（拔掉網路線）使用**，檔案絕對不外流。

3. **🧠 智慧解析與對照記憶**
   * **N-Slot 動態偵測**：自動尋找名冊中含有「時段/社團/課程/節」的欄位。
   * **自動記憶設定**：透過瀏覽器的 `localStorage` 自動記住您設定過的「社團 ── 星期」對照偏好，下次開啟同瀏覽器會自動預填，免去重複選擇的負擔。
   * **公式與特殊字型支援**：底層使用健全的解析機制，完美相容 Excel 的公式計算儲存格與富文本格式。

4. **💅 延平專屬美學排版**
   * **經典延平藍**：產出的 Excel 週課表表頭採用延平藍底色搭配白字置中，極具官方質感。
   * **自動欄寬調整**：精確計算中英文長度，自動拉合合適的欄寬，防止出現 `###` 或文字被截斷。
   * **動態星期剪裁**：若全校沒有人選修週六、週日的課，系統會自動在輸出時隱藏該欄位，讓列印版面更精簡。
   * **自動套用篩選器**：首行自動附帶 Excel 漏斗篩選器（AutoFilter），方便導師直接篩選各班課表。

## 📖 使用步驟說明

### 步驟一：開啟網頁與上傳
打開工具網頁，將從系統下載的 **「錄取名冊 Excel 檔案」** 拖曳至中間的虛線框內（或點擊該區塊手動選取檔案）。
*(注意：Excel 檔案中必須包含一個名為「總表」的工作表。)*

### 步驟二：確認社團對照與上課星期
系統會自動列出本次名冊中出現的所有社團：
* **修改顯示名稱**：可以直接點擊社團名稱的輸入框，修改最後輸出在課表上的名稱（例如：將 `週五舞蹈社` 改為 `舞蹈社`）。
* **設定星期**：為每個社團選擇它在星期幾上課。

### 步驟三：一鍵下載
* 確認完畢後，點擊右下角的 **「⚡ 開始轉換並下載課表」**。
* 系統會於瞬間完成轉檔，並自動觸發瀏覽器下載彙整後的 Excel 檔案（預設檔名為 `原檔名_週一至週五彙整.xlsx`）。

## ⌨️ 鍵盤快速鍵支援
為了加速行政操作，在「步驟二（對照設定）」畫面中，支援以下快速鍵：
* **`Enter`**：直接送出轉換並觸發下載。
* **`Esc`**：返回上一步重新選取檔案。

## 🔧 常見問題排除 (FAQ)

#### Q1：為什麼我點擊「開始轉換並下載」，卻沒有檔案跑出來？
**Ans**：部分瀏覽器基於安全性限制會阻擋「多個檔案自動下載」。請注意瀏覽器網址列右側是否有跳出「已阻擋彈出視窗」的提示，點選「允許」即可。

#### Q2：怎麼清除瀏覽器的記憶設定？
**Ans**：對照紀錄是儲存在您的本機瀏覽器快取中。若您想重置，可以至瀏覽器設定中清除該網頁的 `Site Data`，或改用瀏覽器的「無痕模式（私密瀏覽）」開啟網頁，就不會載入先前的記憶。
