# 🏫 台北市延平國小社團課表整理工具 (網頁版)

這是一個專門為了優化「社團與暑假社團錄取名冊」整理流程而開發的純前端行政輔助工具。  
旨在將傳統混亂的多欄位錄取名冊或網頁名冊，自動重組並美化為以「學生」為單位的個人週課表，大幅節省行政同仁手動複製貼上的排版時間。

* 🌐 **免安裝線上版網址**：[https://rich0512.github.io/excel-web-tool/](https://rich0512.github.io/excel-web-tool/)
* 💼 **適用對象**：延平國小社團課務行政人員、導師及相關處室。

---

## 🌟 核心特色

1. **🚀 雙重任務情境：Excel 檔案與網頁複製貼上**
   * **📂 方案一：上傳 Excel 檔案 (適用於：已有錄取名單，純排課表階段)**
     * 如果您已經有系統導出的整份錄取名冊 Excel（可能包含數十個社團的工作表），這是最快速的方案。
     * **1 秒鐘一鍵上傳**，系統便會自動掃描所有社團及學生，將多個 Sheet 合併並重組，產出個人週課表總表。
   * **📋 方案二：貼上網頁複製名單 (適用於：尚未抽籤，需線上抽籤排課階段)**
     * 如果您需要進行**抽籤**，請在此分頁直接貼上從選課系統網頁上「反黑複製」的學生名冊。
     * 支援輸入正取名額、設定年級限制、並自訂抽籤權重（如高年級優先或低年級優先）。
     * 系統會自動進行**隨機抽籤**、分配籤序並過濾資格，最後一鍵打包下載「週課表總表」及「各社團中籤結果工作表」。

 2. **🧠 智慧特徵推斷與即時預覽**
    * 在貼上名冊時，即使沒有選取到標頭，系統也能自動識別「班級、座號、姓名」。
    * 會主動過濾並安全忽略「學號、選課階段、抽籤順序、備註」等無關資料，長度大於 2 位數的學號也不會被誤判為座號，防錯機制極為強大。
    * **即時解析回饋**：當貼上文字時，網頁會立即動態提示已偵測到的學生筆數與各個關鍵欄位是否正確解析，提升使用信心。

 3. **🚨 即時衝突與衝堂警示區 (Live Conflict Detection)**
    * **多社團交叉比對**：系統在您為 Excel 設定上課星期，或是貼上載入社團的瞬間，會自動在瀏覽器本機進行重疊課表交叉比對。
    * **無侵入式即時列表**：若有學生在同一天重複被兩家社團錄取（衝堂），下方會立刻彈出紅色的警示框，詳細列出班級、座號、姓名及衝突社團，讓您在下載 Excel 前就能即時調整。

 4. **💾 自訂 Excel 檔名與重設機制**
    * 在兩個分頁的下載按鈕上方皆提供「自訂檔名設定 (選填)」輸入框，同仁可自由指派輸出的檔名（例如：115學年度第一學期社團課表），方便建檔與發送。

 5. **🧠 智慧對照記憶**
    * 透過瀏覽器的 `localStorage` 自動記住您設定過的「社團 ── 星期」對照偏好，下次開啟同瀏覽器會自動預填，免去重複選擇的負擔。

 6. **💅 延平專屬美學排版**
    * **經典延平藍**：產出的 Excel 週課表表頭採用延平藍底色搭配白字置中，極具官方質感。
    * **自動欄寬調整與斑馬紋**：防止出現 `###` 或文字被截斷，並附帶交替斑馬紋方便閱讀。
    * **自動套用篩選器**：首行自動附帶 Excel 漏斗篩選器（AutoFilter），方便導師直接篩選各班課表。

 7. **🗳️ 智慧抽籤與年級權重系統 (Lottery Pipeline)**
    * **高/低年級優先權**：支援高年級優先或低年級優先，權重精準排序（如 6 ➔ 5 ➔ ... ➔ 新生）。
    * **年級範圍限制**：限制社團僅能由特定年級報名，超出範圍者自動標記不符資格。
    * **視覺化抽籤結果**：除了產出「週課表總表」外，還會依社團分別產出獨立抽籤結果工作表。正式錄取的學生依「班級、座號」排序，並自動發放籤序；不同年級的籤序儲存格會自動塗上柔和的**馬卡龍粉彩色系**區分，右側亦會自動合併顯示各年級的總錄取人數，一目了然！

## 📖 使用步驟說明

### 📂 模式 A：Excel 檔案匯入
1. 將從系統下載的 **「錄取名冊 Excel 檔案」** 拖曳至上傳區（或點擊手動選取）。
2. 系統會自動辨識並列出所有社團：
   * **修改名稱**：可以直接修改最後輸出在課表上的簡稱（如：將 `週五舞蹈社` 改為 `舞蹈社`）。
   * **設定星期**：為每個社團選擇它在星期幾上課。
3. 點擊 **「⚡ 開始轉換並下載課表」** 即可完成下載。

### 📋 模式 B：直接貼上名單 (網頁複製)
1. 在網頁版點擊上方分頁切換至 **「📋 直接貼上名單」** 模式。
2. 根據該社團之規定，設定招收名額、限制年級及排序優先權。
3. 將選課網頁上的名冊「完整拖曳反黑選擇 (包含欄位標題)」並複製，貼上到文字框中。
   *(不論您有沒有複製到「選課階段、學號」等欄位，系統皆會智慧對齊與過濾；名冊內也不需要先有「抽籤順序」欄位)*
4. 輸入社團名稱、選擇上課星期，點擊 **「➕ 載入此社團名單」**。
5. 如有多個社團需抽籤，請**一次性重複上述步驟載入所有社團**。
6. 確認所有社團皆列在已載入清單後，點擊 **「⚡ 開始彙整並下載課表」** 一鍵打包輸出。

## 🛡️ 資安與個資保護說明

本工具在設計上將**個資保護與安全防範**列為最高優先級，具備以下資安規格：

1. **🔒 100% 本地沙箱執行 (Client-side Sandboxing)**
   * 所有 Excel 解析、隨機抽籤、課表彙整與下載均在您瀏覽器的本機記憶體中完成，**絕不上傳至任何外部伺服器或資料庫**。
   * 受限於瀏覽器沙箱機制，網頁程式**無權讀取**您電腦硬碟中的其他檔案，亦無法修改作業系統設定，確保本機主機的實體安全。

2. **📦 第三方套件完全本地化 (No Third-party CDN Connections)**
   * 本專案已將核心解析庫 `ExcelJS` **完整下載並本地化儲存**在 `js/libs/exceljs.min.js` 中。
   * 系統運作時**完全不請求外部 CDN 網路**，有效防範第三方供應鏈被劫持（Supply Chain Attack）的個資外洩風險，並支援**100% 離線（斷網）使用**。

3. **🧹 關閉即抹除 (Zero Footprint)**
   * 當行政同仁關閉瀏覽器分頁或點擊「繼續轉換新檔案」時，記憶體中暫存的所有學生名冊與設定將會被瀏覽器**自動資源回收並徹底抹除**，絕不殘留於本機系統。

## 👨‍💻 技術與架構說明
本專案採用**高維護性、模組化架構**進行設計，將功能解耦為以下模組：
* `js/libs/exceljs.min.js`：本地端 Excel 核心處理庫（已下載本地封裝）。
* `js/utils.js`：共用字串分析與 LocalStorage 存取工具。
* `js/parser.js`：剪貼簿文字智慧分析與特徵識別引擎。
* `js/processor.js`：學生課表橫向聯結的核心運算核心與隨機抽籤演算法。
* `js/mapping-ui.js`：負責處理多樣模式下的設定表格動態 DOM 生成。
* `js/exporter.js`：Excel 檔案樣式、自動欄寬與斑馬紋樣式包裝。
* `js/app.js`：主要頁面控制器與生命週期管理（包括 beforeunload 安全保護）。
